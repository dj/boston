Boston Maps

What is this?

A collection of choropleth maps of Boston. The current maps are:

  • Unemployment Map
  • Household Income Map
  • Rent Burden Map
  • Travel Time To Work
  • Homeownership Map

Who did this?

DJ Hartman for ARTG 5330 Vizualization Technologies with Siqi Zhu.

Can I use this with my cities data? Where is the data from?

Absolutely. Checkout the repository, or get in contact with me, I'd love to help you out. The code is MIT Licensed.
